# Hello World Python Application

A simple Python hello world application with proper project structure and tests.

## Project Structure

```
.
├── src/
│   ├── __init__.py
│   └── main.py          # Main application code
├── tests/
│   ├── __init__.py
│   └── test_main.py     # Unit tests
├── requirements.txt      # Project dependencies
└── README.md            # This file
```

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```bash
python src/main.py
```

## Testing

Run tests:
```bash
pytest
```

Run tests with verbose output:
```bash
pytest -v
```
